紹介文

平成元年 (1989年) 生まれです。
早稲田大学 基幹理工学部 電子光システム学科 卒業 → 東京大学大学院 工学系研究科 電気系工学専攻 中退
新卒で入社したマーケティングリサーチツールベンダー企業にて営業と人事を担当。
もし、ご興味があればお気軽にメールもしくはTwitterのDMにてご連絡ください。

  • ドメイン知識： 前職のマーケティングリサーチ
  • アウトプット： GitHub・HackMD・Qiitaを御覧ください。
これから

Webエンジニア（サーバーサイド寄り）として転職を希望しております。
これまでPHP→Laravelをメインで学習してきましたが、RubyやPython、はたまたGo等の静的型付け言語等でも構いません。
事業領域に関しては、現時点ではEdTech・HRTech領域への興味関心が強いです。
実際に学生時代には教育関連のNPOにインターンやボランティアとして活動していました。

また、Djangoを扱う際に基本的な文法だけは習得しました。
余談ですが、環境はAnacondaを使用せず、pipとvenvでPython3.8を使用しています。

Firebase

Docker

 ★★★☆☆ チーム開発ではLaravelとVue.jsを使用しており、開発環境にはLaradockを選択しました。
開発環境構築のために私がLaradockの知見を共有し開発を進めておりました。
基礎知識はUdemyの入門動画で学習し、実際の開発ではDocker Composeも使用していました。
学習しても底が知れないと感じたので、一旦学習をストップしました。

MySQL

 ★★★☆☆ MySQLというよりDB設計です。「 おうちで学べるデータベースのきほん」という書籍を一通り読みました。
物理設計は出来ないですが、仕様から正規化されたエンティティで構成されたER図を作成することは出来ます。
実際のチーム開発では、私がMySQL Workbenchにて作成したER図を基に開発を進めておりました。

git
